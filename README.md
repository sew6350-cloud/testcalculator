# GitHub Pages Demo - Instant Calculator

A simple interactive calculator built with HTML, CSS, and JavaScript.

## How to deploy this on GitHub Pages

1. Create a new repository on GitHub
2. Upload `index.html` to the repository
3. Go to **Settings** > **Pages**
4. Under "Source", select **Deploy from a branch**
5. Choose **main** branch and **/ (root)** folder
6. Click **Save**
7. Wait ~30 seconds, then refresh â€” your live URL will appear at the top

Your site will be live at: `https://[your-username].github.io/[repo-name]`
